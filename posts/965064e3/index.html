ol/style/Style~Style

Style 是矢量要素渲染样式的容器。需要注意的是，使用该样式的要素或图层，在通过set*()方法对样式或其子级所做的任何更改，在要素或图层重新渲染之前都不会立即生效。

构造方法

参数名称 参数类型 描述
geometry string|Geometry|GeometryFuntion|undefined 该参数可以是：要素属性或几何图形或是一个返回此样式渲染的几何体函数。
fill Fill|undefined 填充样式
image ImageStyle|undefined Image 样式，ImageStyle 用于创建子类，而不是在应用程序中实例化。Icon、CircleStyle和RegularShape的基类。
renderer RenderFunction | undefined 自定义渲染器。配置后，填充、描边和Image 样式将被忽略，并且提供的函数将在每个几何体的每个渲染帧中调用。
hitDetectionRenderer RenderFunction | undefined 用于命中检测的自定义渲染器。如果提供，将用于命中检测渲染。
stroke Stroke | undefined 描边样式
text Text | undefined 文本样式
zIndex number | undefined z 索引值

方法

clone

clone() => {Style}

克隆样式。

返回值：克隆的样式。

getFill

getFill() => {Fill | null}

获取填充样式。

返回值：填充样式。

getGeometry

getGeometry() => {string | Geometry | GeometryFunction | null}

获取要渲染的几何体。

返回值：返回将要素属性或几何图形或使用此样式渲染的几何图形的函数。

getGeometryFunction

getGeometryFunction() => {GeometryFunction}

获取用于渲染的几何图形的函数。

返回值：使用要素调用的函数，并返回要渲染的几何图形而不是要素的几何图形。

getHitDetectionRenderer

getHitDetectionRenderer() => {RenderFunction | null}

获取使用setHitDetectionRenderer()方法或hitDetectionRenderer构造函数选项配置的自定义渲染器函数。

返回值：自定义的渲染器函数。

getImage

getImage() => {ImageStyle | null}

获取 Image 样式。

返回值：Image 样式。

getRenderer

getRenderer() => {RenderFunction | null}

获取使用setRenderer或构造函数的renderer选项配置的自定义渲染器函数。

返回值：自定义的渲染器函数。

getStroke

getStroke() => {Stroke | null}

获取描边样式。

返回值：描边样式。

getText

getText() => {Text | null}

获取文本样式。

返回值：文本样式。

getZIndex

getZIndex() => {number | undefined}

获取此样式的z-index值。

返回值：z-index值。

setFill

setFill(fill)

设置填充样式。

参数名称 参数类型 描述
fill Fill | null 填充样式。

setGeometry

setGeometry(geometry)

设置要渲染的几何图形，而不是要素的几何图形。

参数名称 参数类型 描述
geometry string|Geometry|GeometryFuntion 要素属性或几何图形或是一个返回此样式渲染的几何体函数。

setHitDetectionRenderer

setHitDetectionRenderer(renderer)

为命中检测中使用的此样式设置自定义渲染器函数。

参数名称 参数类型 描述
renderer RenderFunction |null 自定义渲染器函数。

setImage

setImage(image)

设置Image 样式。

参数名称 参数类型 描述
image ImageStyle Image 样式。

setRenderer

setRenderer(renderer)

为此样式设置自定义渲染器函数。配置后，填充、描边和 Image 样式将被忽略。

参数名称 参数类型 描述
renderer RenderFunction |null 自定义渲染器函数。

setStroke

setStroke(stroke)

设置描边样式。

参数名称 参数类型 描述
stroke Stroke |null 描边样式。

setText

setText(text)

设置文本样式。

参数名称 参数类型 描述
text Text 文本样式。

setZIndex

setZIndex(zIndex)

设置文本样式。

参数名称 参数类型 描述
zIndex number | undefined z-index值。
